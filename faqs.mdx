## Account and Registration

<ExpandableGroup>
  <Expandable title="How do I create an account?" default-open="false">
    Visit the TrilogyMC website and click "Sign Up". Provide your email, create a username and password, then verify your email to activate your account.
  </Expandable>
  <Expandable title="Is registration free?" default-open="false">
    Yes, TrilogyMC registration is completely free. You get access to all documentation, community features, and basic support at no cost.
  </Expandable>
  <Expandable title="Can I change my username?" default-open="false">
    Username changes are allowed once every 30 days. Contact support via email if you need to change your username outside this window.
  </Expandable>
  <Expandable title="What if I forget my password?" default-open="false">
    Use the "Forgot Password" link on the login page. We'll send a reset link to your registered email address.
  </Expandable>
</ExpandableGroup>

## Server and Technical Questions

<ExpandableGroup>
  <Expandable title="What Minecraft versions do you support?" default-open="false">
    We provide support for all current Minecraft versions, with focus on the latest stable releases. Legacy version support may be limited.
  </Expandable>
  <Expandable title="Do you offer hosting services?" default-open="false">
    TrilogyMC is a community platform and does not provide server hosting. We offer guides and support for self-hosted servers.
  </Expandable>
  <Expandable title="How do I install plugins?" default-open="false">
    Download plugin JAR files and place them in your server's plugins folder. Restart the server to load the plugins.
  </Expandable>
  <Expandable title="Can I contribute to the documentation?" default-open="false">
    Yes! Contact us via Discord or email if you'd like to contribute guides, tutorials, or improvements to our documentation.
  </Expandable>
</ExpandableGroup>

<CodeGroup tabs="Plugin Installation,Server Startup">
  ```bash
  # Plugin installation
  cd plugins/
  wget https://example.com/plugin.jar
  # Restart server
  ```
  ```bash
  # Basic server startup
  java -Xmx4G -Xms4G -jar server.jar nogui
  ```
</CodeGroup>

## Billing and Premium Features

<ExpandableGroup>
  <Expandable title="Are there premium features?" default-open="false">
    TrilogyMC offers optional premium features like priority support and advanced documentation access. These are completely optional.
  </Expandable>
  <Expandable title="How do I upgrade to premium?" default-open="false">
    Visit your account settings and select a premium plan. Payment is processed securely through our payment partners.
  </Expandable>
  <Expandable title="Can I cancel premium anytime?" default-open="false">
    Yes, you can cancel premium subscriptions at any time. You'll retain access until the end of your billing period.
  </Expandable>
</ExpandableGroup>
