## Code Examples

For developers working with TrilogyMC plugins, here's a basic example of setting up a simple plugin:

<CodeGroup tabs="Java,Kotlin">
  ```java
  public class TrilogyPlugin extends JavaPlugin {
      @Override
      public void onEnable() {
          getLogger().info("TrilogyMC Plugin enabled!");
      }
  
      @Override
      public void onDisable() {
          getLogger().info("TrilogyMC Plugin disabled!");
      }
  }
  ```
  ```kotlin
  class TrilogyPlugin : JavaPlugin() {
      override fun onEnable() {
          logger.info("TrilogyMC Plugin enabled!")
      }
  
      override fun onDisable() {
          logger.info("TrilogyMC Plugin disabled!")
      }
  }
  ```
</CodeGroup>
