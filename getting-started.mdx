## Account Registration

<Steps>
  <Step title="Visit Website" icon="globe">
    Navigate to the TrilogyMC website and click on the "Sign Up" button in the top right corner.
  </Step>
  <Step title="Provide Details" icon="edit">
    Enter your email address, create a username, and set a secure password.
  </Step>
  <Step title="Verify Email" icon="check-circle">
    Check your email for a verification link and click it to activate your account.
  </Step>
  <Step title="Complete Profile" icon="user">
    Add optional profile information to personalize your experience.
  </Step>
</Steps>

## Accessing Documentation

Once registered, you can access our full documentation library. Our guides cover everything from basic server setup to advanced plugin development.

<CodeGroup tabs="Command Line,Web Interface">
  ```bash
  # Clone our example repository
  git clone https://github.com/trilogymc/examples.git
  cd examples
  
  # Run the setup script
  ./setup.sh
  ```
  ```javascript
  // Web interface example
  const trilogyApi = new TrilogyAPI({
      apiKey: 'your-api-key',
      baseUrl: 'https://api.trilogymc.com'
  });
  
  trilogyApi.getGuides().then(guides => {
      console.log('Available guides:', guides);
  });
  ```
</CodeGroup>

## First Server Setup

<Expandable title="Basic Server Configuration" default-open="false">
  Setting up your first Minecraft server with TrilogyMC support is easy. Follow these steps to get started:
  
  1. Download the latest Minecraft server JAR from minecraft.net
  2. Create a new directory for your server
  3. Place the JAR file in the directory and run it once to generate configuration files
  4. Edit server.properties to configure basic settings
  5. Install recommended plugins from our resources section
  6. Start the server and connect to test your setup
</Expandable>
